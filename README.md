
## Como Rodar

### Backend (porta 3001)
```bash
cd backend
npm install
npm run start:dev
```

### Frontend (porta 3000)
```bash
cd frontend
npm install
npm run dev
```

Acesse: http://localhost:3000


Para rodar o serviço Python (terceiro terminal):
bashcd ml
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
